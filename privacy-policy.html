Privacy Policy

Meta Inspector Pro — Chrome Extension · Last updated: February 2026

TL;DR: Meta Inspector Pro does NOT collect, store, transmit, or share any user data. All analysis happens locally in your browser.

What This Extension Does

Meta Inspector Pro analyzes the metadata (meta tags, Open Graph, Twitter Card, JSON-LD structured data, link tags) of the web page you are currently viewing. It displays this information in a convenient popup for developers, SEO specialists, and marketers.

Data Collection

This extension does not collect any data. Specifically:

  • No personal information is collected
  • No browsing history is tracked or stored
  • No data is sent to any remote server
  • No analytics, telemetry, or tracking scripts are used
  • No cookies are set or read by the extension
  • No third-party services are integrated

How It Works

When you click the extension icon, a content script reads the HTML metadata of the currently active tab. This data is processed entirely within your browser and displayed in the extension popup. No data leaves your device.

Permissions Explained

  • activeTab — Allows the extension to read the content of the tab you're currently viewing, only when you click the extension icon. No background access.
  • scripting — Required to inject the content script that reads metadata from the current page.

Data Storage

The extension does not persistently store any data. The metadata displayed in the popup exists only in memory for the duration of the popup session and is discarded when the popup is closed.

The JSON export feature saves data to a file on your local device at your explicit request. This file is never transmitted anywhere.

Third-Party Services

This extension does not use any third-party services, APIs, SDKs, or analytics platforms.

Changes to This Policy

If this privacy policy is updated, the changes will be reflected on this page with an updated date. The extension will continue to operate with the same privacy-first principles.

Contact

If you have any questions about this privacy policy, please open an issue on the extension's GitHub repository or contact the developer.
